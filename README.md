# To-Done

Not a task list, a view of actions that you will take today.

## To-Done Design Principles

To-Done design is based on organic imperfection and it's purpose is to try to create the look and feel of a real notebook.

    •	Single Theme (Notebook Paper)
    •	Handwritten personality
    •	Warm colors
    •	Minimal shadows
    •	Organic imperfection (light rotation and light variation)
